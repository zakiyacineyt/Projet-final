# Premium To-Do

Une application de gestion de tâches moderne et élégante avec une interface premium.

## Fonctionnalités

- **Création de tâches** : Ajoutez des tâches avec titre, catégorie, priorité, date d'échéance et tags
- **Gestion des priorités** : Classez vos tâches en priorité haute, moyenne ou basse
- **Catégories** : Organisez vos tâches par catégorie (Work, Personal, Wellbeing, Finance)
- **Filtres** : Filtrez par statut, priorité ou catégorie
- **Recherche** : Recherchez rapidement dans vos tâches
- **Glisser-déposer** : Réorganisez vos tâches par drag & drop
- **Duplication** : Dupliquez une tâche avec Ctrl + Drag
- **Statistiques** : Visualisez votre progression avec des statistiques détaillées
- **Mode sombre** : Interface adaptée avec un thème noir mat profond
- **Animations** : Transitions fluides et animations d'apparition
- **Persistance** : Sauvegarde automatique dans le localStorage
- **Annulation** : Possibilité d'annuler la suppression d'une tâche

## Raccourcis clavier

| Raccourci | Action |
|-----------|--------|
| `Enter` | Ajouter une tâche |
| `Delete` | Supprimer la tâche sélectionnée |
| `Ctrl + Drag` | Dupliquer une tâche |

## Technologies

- HTML5
- CSS3 (Variables CSS, Flexbox, Grid, Animations)
- JavaScript (ES6+)
- LocalStorage pour la persistance des données

## Installation

1. Clonez ou téléchargez le projet
2. Ouvrez `index.html` dans un navigateur moderne
